**Human explanation pack:**
- **Issue:** An all-true boolean expression (e.g. `x | ~x`) makes `PhaseOracleGate`/`BitFlipOracleGate` raise `TranspilerError: HighLevelSynthesis is unable to synthesize "Phase Oracle"`.
- **Root cause:** A tautology becomes a single ESOP clause of all `-` (no controls). The old code did `zip(*[])` and tried to unpack two values from it, which fails.
- **Expected behavior:** phase oracle for `f(x)=1` is `-I` (global phase ╧Ç); bit-flip oracle for `f(x)=1` flips the output qubit unconditionally.
- **Fix:** detect the empty-control clause and emit `global_phase += pi` (phase) or `x(output)` (bit-flip); all other clauses use the existing controlled-gate path unchanged.
- **Proof:** operator equals `-I` / unconditional `X`; regression tests fail before and pass after.
